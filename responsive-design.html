Reflections

In this assignment, I picked my color palette based on the background image I used in the header with Adobe Color. This ensures that there is some sort of color consistency — and as a bonus, it allowed me to make a nice background color transition between the header to the "advantages" section. Likewise, my color palette is also applied to the graphic element in the header for consistency.

The font pairing is contrasting to make the headlines stand out more with the use of a serif font and a sans serif font, similarly to the previous assignment.

While the name of the assignment is responsive design, I could've made a burger menu for smaller screens, instead of just using display: hidden; in a media query. However, I decided to just settle on that so that the navigation doesn't overflow on smaller screens.
