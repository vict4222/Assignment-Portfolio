Reflections

In this assignment, I styled the landing page with the classes provided by Tailwind CSS and used their docs to get through the assignment. I didn't choose a font myself to use this time, so I went along with the default font provided by Tailwind CSS. My color palette is again based on the background image in the header, but I used the color classes as an exercise to get familiar with Tailwind CSS.

Tailwind CSS, while complex and at the same time powerful, was perhaps overkill for this assignment. It has its own drawbacks, such as reducing the readability of the HTML document due to the sheer amount of classes, which makes it difficult to navigate in the HTML document.

It is also contradicting the DRY method, which is "don't repeat yourself". To style all similar elements the same, you would have to copy paste all the classes. If you wanted to change the styling of h2 elements, you would have to change all classes for h2 elements accordingly. But I have barely scratched the surface of Tailwind CSS, so maybe there are better ways of dealing with that.
